# Word to InDesign インポートスクリプト

Word文書（.docx）をInDesignにインポートし、段落スタイルを自動変換するExtendScript（JSX）です。

## 対応環境

- InDesign 2025

## 機能概要

1. **Word文書のインポート**: 選択したWord文書をInDesignドキュメントに配置
2. **段落スタイルの自動変換**: Word側の段落スタイルをInDesign側の段落スタイルにマッピング
3. **自動ページ追加**: テキストがオーバーフローした場合、マスターページを使用して見開きページを自動追加
4. **フォント置換**: 存在しないフォントを自動的に置換
5. **既存コンテンツの保護**: インポートしたストーリーのみに処理を適用し、既存のドキュメント内容は影響を受けない

## 段落スタイルマッピング

| Word側 | InDesign側 | 備考 |
|--------|------------|------|
| 大項目 | 大見出し1 | 先頭の■記号を自動削除 |
| 小項目 | 小項目 | 先頭に□記号を自動追加（なければ） |
| 標準 | Normal | |
| 演習タイトル | 演習タイトル | |
| 図表番号 | 図番号 | |
| リスト | リスト | 同名でも明示的に適用 |
| 番号 | 番号リスト | |

## フォント置換

| 置換元 | 置換先 |
|--------|--------|
| MS明朝 Bold | BIZ UDゴシック Regular |

## 使用方法

1. InDesignで対象のドキュメントを開く
2. マスターページ「H-本文マスター」が存在することを確認
3. スクリプトパネルから `ImportWordToInDesign.jsx` を実行
4. ファイル選択ダイアログでWord文書（.docx）を選択
5. 確認ダイアログで「OK」をクリック
6. 処理完了後、結果が表示される

## 前提条件

- InDesignドキュメントに以下が設定されていること：
  - マスターページ「H-本文マスター」（見開き）
  - 段落スタイル：大見出し1、小項目、Normal、演習タイトル、図番号、リスト、番号リスト

## 処理結果

スクリプト実行後、以下の情報が表示されます：

- 既存ページ数
- 追加された見開き数
- インポートされた段落数
- スタイル変換件数
- 小項目への□記号追加件数
- フォント置換件数
- 処理時間

## 注意事項

- スクリプト実行前に、InDesignドキュメントのバックアップを取ることを推奨します
- Word文書内の画像は別途配置が必要な場合があります
- 処理中は画面の再描画が無効化されるため、大きなファイルでは時間がかかる場合があります
